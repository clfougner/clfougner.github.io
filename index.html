Analyzing mutational signatures in mice using deconstructSigs

Mutational signatures were pioneered by Nik-Zainal et al. in 2012 and the concept has since become instrumental in the analysis of the etiology of cancer. This blog post shows how a mutational signature analysis can be performed using the deconstructSigs R package, and the adjustments to the source code that need to be made in order to be able to do so on tissue from mice.
